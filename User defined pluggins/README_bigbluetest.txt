README file for the Big Blue Test pluggin
Authors: Kendall Weihe & Steve Pearce
----------------------
Please post this pluggin on the public CS 485 web page
Usage:
    big_blue_test
Description:
   This pluggin adds a built-in command that will play a simple game to test your knowledge of UK basketball player's jersey numbers.  It can be called any number of times.  Any parameters passed to the built-in command will be ignored.
-----------------------
