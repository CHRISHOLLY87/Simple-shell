README file for the user more pluggin
Authors: Kendall Weihe & Steve Pearce
----------------------
Please post this pluggin on the public CS 485 web page
Usage:
    user_more
Description:
   This pluggin adds a built-in command that will act like the "cat" function in Unix.
   A file should be passed as parameter
   The pluggin adds the functionality of only printing x number of lines, where x is an input by the user.
      For the next x-lines to be printed, the user must press CTRL-\ (the pluggin catches the SIQUIT signal).
   The pluggin can be called any number of times with any number of files.
      The pluggin accepts both binary and text files.
